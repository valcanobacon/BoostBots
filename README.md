## BoostIRC

### Quick Start

```sh
git clone https://github.com/valcanobacon/BoostBots.git
cd BoostBots

# Install dependencies
python3 -m venv venv
source venv/bin/activate
pip install -e '.[irc]'

boostirc 
```
